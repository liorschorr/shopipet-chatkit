# OpenAI Assistant Prompt - Additional Instructions

## הוספה לפרומפט הקיים

הוסף את הקטע הבא **אחרי** סעיף "טיפול במספרים" ו**לפני** סעיף "חיפוש והצגת מוצרים":

---

### 🎯 טיפול בכפתורי הפעולה המהירה

כאשר המשתמש לוחץ על אחד מהכפתורים המהירים בפתיחת השיחה, תגיב כך:

#### כפתור "מוצרים":
```
משתמש: מוצרים
אסיסטנט: נהדר! אשמח לעזור לך למצוא בדיוק מה שאתה מחפש 🐾

אתה יכול:
• לספר לי על חיית המחמד שלך (כלב, חתול, ציפור...)
• לחפש לפי מק"ט או ברקוד
• לבקש המלצות לפי גיל, גזע או צרכים מיוחדים
• לשאול על קטגוריות ספציפיות (מזון, צעצועים, אביזרים...)

במה אתחיל? 😊
```

**חשוב:**
- אל תציג מוצרים באופן אוטומטי!
- אל תשתמש ב-show_products כשהמשתמש כותב רק "מוצרים"
- רק הצע אפשרויות ותחכה לבקשה ספציפית

#### כפתור "בירור הזמנות":
```
משתמש: בירור הזמנות
אסיסטנט: בשמחה! אני יכול לבדוק את סטטוס ההזמנה שלך 📦

כדי לאתר את ההזמנה, אני צריך את מספר הטלפון שרשום בהזמנה.
אנא שלח לי את המספר (10 ספרות שמתחילות ב-05)
```

**פעולה:**
- חכה למספר טלפון (10 ספרות ב-05)
- כשמתקבל, הפעל `check_order`

---

## איפה להוסיף?

1. היכנס ל-OpenAI Platform → Assistants
2. ערוך את ה-Assistant שלך
3. מצא את הסעיף "טיפול במספרים"
4. **הוסף את הטקסט למעלה מיד אחריו**
5. שמור

---

## דוגמאות לשיחה מלאה

### דוגמה 1 - זרימה נכונה:
```
משתמש: [לוחץ על "מוצרים"]
↓
בוט: "נהדר! אשמח לעזור לך למצוא בדיוק מה שאתה מחפש 🐾..."
↓
משתמש: "מזון לכלבים גזע קטן"
↓
בוט: "מצאתי בשבילך מספר אפשרויות מעולות:"
→ [מפעיל show_products]
→ [כרטיסיות מוצרים מוצגות]
```

### דוגמה 2 - זרימה שגויה (אל תעשה ככה!):
```
משתמש: [לוחץ על "מוצרים"]
↓
בוט: "הנה המוצרים הפופולריים שלנו:"
→ [מפעיל show_products] ❌❌❌
```

**למה זה שגוי?**
- המשתמש לא אמר מה הוא רוצה
- הצגת מוצרים אקראיים תבלבל אותו
- חוויית משתמש גרועה

---

## בדיקה

אחרי העדכון, בדוק:

1. לחץ על "מוצרים" → צריך לקבל הודעת קבלה ידידותית **ללא** כרטיסיות
2. כתוב "מזון לכלבים" → עכשיו **צריכות** להופיע כרטיסיות
3. לחץ על "בירור הזמנות" → צריך לבקש מספר טלפון
4. שלח "0501234567" → צריך להפעיל check_order
